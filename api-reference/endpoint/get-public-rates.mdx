api: "GET /public/rates"
description: "Retrieve current exchange rates for supported currencies"
---

## Overview

Retrieve current exchange rates for supported currencies

## Response Codes

### Success Responses

<ResponseField name="status" type="number">
  HTTP status code: 200
</ResponseField>

<ResponseField name="message" type="string">
  Exchange rates retrieved successfully
</ResponseField>

<ResponseField name="data" type="array">
  Array of exchange rate objects
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET 'https://api.Hashrails.xyz/v1/public/rates'
```

```javascript JavaScript
const response = await fetch('https://api.Hashrails.xyz/v1/public/rates');

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get('https://api.Hashrails.xyz/v1/public/rates')

data = response.json()
print(data)
```

</CodeGroup>
