## Overview

Check the health status of the API service

## Response Codes

### Success Responses

<ResponseField name="status" type="number">
  HTTP status code: 200
</ResponseField>

<ResponseField name="message" type="string">
  Service is healthy
</ResponseField>

<ResponseField name="data" type="object">
  Health check data (HealthCheck)
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET 'https://api.Hashrails.xyz/v1/public/health'
```

```javascript JavaScript
const response = await fetch('https://api.Hashrails.xyz/v1/public/health');

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get('https://api.Hashrails.xyz/v1/public/health')

data = response.json()
print(data)
```

</CodeGroup>

## Example Response

### Success Response

```json
{
  "status": true,
  "service": "HashRails API service"
}
```
