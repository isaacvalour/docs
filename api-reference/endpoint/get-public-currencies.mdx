## Overview

Retrieve list of supported currencies

## Response Codes

### Success Responses

<ResponseField name="status" type="number">
  HTTP status code: 200
</ResponseField>

<ResponseField name="message" type="string">
  Currencies retrieved successfully
</ResponseField>

<ResponseField name="data" type="array">
  Array of currency objects
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET 'https://api.Hashrails.xyz/v1/public/currencies'
```

```javascript JavaScript
const response = await fetch('https://api.Hashrails.xyz/v1/public/currencies');

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get('https://api.Hashrails.xyz/v1/public/currencies')

data = response.json()
print(data)
```

</CodeGroup>
