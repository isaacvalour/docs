api: "GET /public/banks/lookup"
description: "Lookup bank information by account number"
---

## Overview

Lookup bank information by account number

## Request

<ParamField query="account_number" type="string" required>
  Bank account number to lookup
  
  **Example:** `1234567890`
</ParamField>

## Response Codes

### Success Responses

<ResponseField name="status" type="number">
  HTTP status code: 200
</ResponseField>

<ResponseField name="message" type="string">
  Bank lookup successful
</ResponseField>

<ResponseField name="data" type="object">
  Bank lookup data
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET 'https://api.Hashrails.xyz/v1/public/banks/lookup?account_number=1234567890'
```

```javascript JavaScript
const response = await fetch('https://api.Hashrails.xyz/v1/public/banks/lookup?account_number=1234567890');

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get('https://api.Hashrails.xyz/v1/public/banks/lookup', params={
    'account_number': '1234567890'
})

data = response.json()
print(data)
```

</CodeGroup>
