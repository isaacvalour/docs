## Overview

Retrieve list of supported banks

## Response Codes

### Success Responses

<ResponseField name="status" type="number">
  HTTP status code: 200
</ResponseField>

<ResponseField name="message" type="string">
  Banks retrieved successfully
</ResponseField>

<ResponseField name="data" type="array">
  Array of bank objects
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET 'https://api.Hashrails.xyz/v1/public/banks'
```

```javascript JavaScript
const response = await fetch('https://api.Hashrails.xyz/v1/public/banks');

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get('https://api.Hashrails.xyz/v1/public/banks')

data = response.json()
print(data)
```

</CodeGroup>
