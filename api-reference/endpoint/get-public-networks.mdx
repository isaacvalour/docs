title: "Get Supported Networks"
api: "GET /public/networks"
description: "Retrieve list of supported blockchain networks"
---

## Overview

Retrieve list of supported blockchain networks

## Response Codes

### Success Responses

<ResponseField name="status" type="number">
  HTTP status code: 200
</ResponseField>

<ResponseField name="message" type="string">
  Networks retrieved successfully
</ResponseField>

<ResponseField name="data" type="array">
  Array of network objects
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET 'https://api.Hashrails.xyz/v1/public/networks'
```

```javascript JavaScript
const response = await fetch('https://api.Hashrails.xyz/v1/public/networks');

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get('https://api.Hashrails.xyz/v1/public/networks')

data = response.json()
print(data)
```

</CodeGroup>
