## Overview

Retrieve list of public recipients

## Response Codes

### Success Responses

<ResponseField name="status" type="number">
  HTTP status code: 200
</ResponseField>

<ResponseField name="message" type="string">
  Recipients retrieved successfully
</ResponseField>

<ResponseField name="data" type="array">
  Array of recipient objects
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET 'https://api.Hashrails.xyz/v1/public/recipients'
```

```javascript JavaScript
const response = await fetch('https://api.Hashrails.xyz/v1/public/recipients');

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get('https://api.Hashrails.xyz/v1/public/recipients')

data = response.json()
print(data)
```

</CodeGroup>
