## Overview

Retrieve list of public virtual accounts

## Response Codes

### Success Responses

<ResponseField name="status" type="number">
  HTTP status code: 200
</ResponseField>

<ResponseField name="message" type="string">
  Virtual accounts retrieved successfully
</ResponseField>

<ResponseField name="data" type="array">
  Array of virtual account objects
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET 'https://api.Hashrails.xyz/v1/public/virtual-accounts'
```

```javascript JavaScript
const response = await fetch('https://api.Hashrails.xyz/v1/public/virtual-accounts');

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get('https://api.Hashrails.xyz/v1/public/virtual-accounts')

data = response.json()
print(data)
```

</CodeGroup>
