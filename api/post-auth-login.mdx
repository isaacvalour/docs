## Overview

Authenticate user with email and password, optionally with 2FA

## Request

<ParamField body="email" type="string" required>
  User email address
  
  **Example:** `user@example.com`
</ParamField>

<ParamField body="password" type="string" required>
  User password
  
  **Example:** `StrongP@ssw0rd!`
</ParamField>

## Example Request

<CodeGroup>

```bash cURL
curl -X POST \
  'https://api.railsfromthecrypt.com/v1/auth/login' \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -d '{"email": "user@example.com", "password": "StrongP@ssw0rd!"}'
```

```javascript JavaScript
const response = await fetch('https://api.railsfromthecrypt.com/v1/auth/login', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_TOKEN'
  },
  body: JSON.stringify({"email": "user@example.com", "password": "StrongP@ssw0rd!"})
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

url = 'https://api.railsfromthecrypt.com/v1/auth/login'
headers = {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_TOKEN'
}
data = {"email": "user@example.com", "password": "StrongP@ssw0rd!"}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(result)
```
