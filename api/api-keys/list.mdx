---
title: "List API Keys"
api: "GET /v1/api-keys"
description: "Get list of all API keys for the authenticated user"
---

## Overview

Retrieve a list of all API keys associated with the authenticated user account. API keys are used for programmatic access to the API.

<Note>
API key secrets are never returned in the response for security reasons. Only the key ID, label, and metadata are included.
</Note>

## Query Parameters

<ParamField query="limit" type="number">
  Number of keys to return (default: 20, max: 100)
</ParamField>

<ParamField query="offset" type="number">
  Number of keys to skip (default: 0)
</ParamField>

<ParamField query="status" type="string">
  Filter by key status (`active`, `inactive`, `revoked`)
</ParamField>

<ParamField query="sort" type="string">
  Sort order (`created_at`, `-created_at`, `label`, `-label`)
</ParamField>

## Response

<ResponseField name="api_keys" type="array">
  Array of API key objects
</ResponseField>

<ResponseField name="api_keys[].id" type="string">
  Unique identifier for the API key
</ResponseField>

<ResponseField name="api_keys[].label" type="string">
  Human-readable label for the key
</ResponseField>

<ResponseField name="api_keys[].status" type="string">
  Key status (`active`, `inactive`, `revoked`)
</ResponseField>

<ResponseField name="api_keys[].permissions" type="array">
  Array of permissions granted to this key
</ResponseField>

<ResponseField name="api_keys[].last_used" type="string">
  Last usage timestamp (ISO 8601)
</ResponseField>

<ResponseField name="api_keys[].created_at" type="string">
  Creation timestamp (ISO 8601)
</ResponseField>

<ResponseField name="api_keys[].expires_at" type="string">
  Expiration timestamp (ISO 8601, null if no expiration)
</ResponseField>

<ResponseField name="pagination" type="object">
  Pagination information
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET "https://api.railsfromthecrypt.com/v1/api-keys" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.railsfromthecrypt.com/v1/api-keys', {
  headers: {
    'Authorization': 'Bearer YOUR_ACCESS_TOKEN'
  }
});

const data = await response.json();
```

</CodeGroup>
