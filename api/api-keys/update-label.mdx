---
title: "Update API Key Label"
api: "PATCH /v1/api-keys/{id}/label"
description: "Update the label of an API key"
---

## Overview

Update the label and description of an existing API key.

## Path Parameters

<ParamField path="id" type="string" required>
  Unique identifier for the API key
</ParamField>

## Request Body

<ParamField body="label" type="string" required>
  New label for the API key
</ParamField>

<ParamField body="description" type="string">
  New description for the API key
</ParamField>

## Response

<ResponseField name="id" type="string">
  API key ID
</ResponseField>

<ResponseField name="label" type="string">
  Updated label
</ResponseField>

<ResponseField name="description" type="string">
  Updated description
</ResponseField>

<ResponseField name="updated_at" type="string">
  Update timestamp (ISO 8601)
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X PATCH "https://api.railsfromthecrypt.com/v1/api-keys/ak_123e4567-e89b-12d3-a456-426614174000/label" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "label": "Updated Production Key",
    "description": "Updated description"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.railsfromthecrypt.com/v1/api-keys/ak_123e4567-e89b-12d3-a456-426614174000/label', {
  method: 'PATCH',
  headers: {
    'Authorization': 'Bearer YOUR_ACCESS_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    label: 'Updated Production Key',
    description: 'Updated description'
  })
});

const result = await response.json();
```

</CodeGroup>
