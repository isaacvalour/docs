## Overview

Retrieve notification counts grouped by status and type. This is useful for displaying notification badges and counts in the UI.

## Usage Examples

### Display Notification Badge

<CodeGroup>

```javascript JavaScript
async function getNotificationBadge() {
  try {
    const response = await fetch('https://api.railsfromthecrypt.com/v1/notifications/count', {
      headers: {
        'Authorization': `Bearer ${accessToken}`
      }
    });

    const counts = await response.json();
    
    // Update UI with notification counts
    updateNotificationBadge(counts.unread);
    updateNotificationCounts(counts);
    
    return counts;
  } catch (error) {
    console.error('Error fetching notification counts:', error);
    throw error;
  }
}

function updateNotificationBadge(unreadCount) {
  const badge = document.getElementById('notification-badge');
  if (unreadCount > 0) {
    badge.textContent = unreadCount;
    badge.style.display = 'block';
  } else {
    badge.style.display = 'none';
  }
}
```

</CodeGroup>
