## Overview

Check the health and status of the HR API system. This endpoint provides information about the system's current state, including database connectivity, external service status, and overall system health.

<Note>
  This endpoint does not require authentication and can be used for monitoring and load balancer health checks.
</Note>

## Request

No parameters required.

## Status Codes

<ResponseField name="200" type="number">
  Health check successful (regardless of system status)
</ResponseField>

<ResponseField name="503" type="number">
  Service unavailable (system is unhealthy)
</ResponseField>

## Status Meanings

### System Status

<CardGroup cols={3}>
  <Card title="Healthy" icon="check-circle">
    All services are functioning normally

    - Database: \< 100ms response time
    - Redis: \< 10ms response time
    - External APIs: \< 1000ms response time
    - Memory usage: \< 80%
  </Card>
  <Card title="Degraded" icon="exclamation-triangle">
    Some services are experiencing issues

    - One or more services have slow response times
    - Memory usage: 80-90%
    - System is still functional but may be slow
  </Card>
  <Card title="Unhealthy" icon="times-circle">
    Critical services are down

    - Database or Redis is unavailable
    - Memory usage: \> 90%
    - System is not functional
  </Card>
</CardGroup>

## Monitoring Integration

### Load Balancer Health Check

<CodeGroup>

```bash Load Balancer Configuration
# Nginx configuration
upstream hr_api {
    server api1.hashrails.xyz:443;
    server api2.hashrails.xyz:443;
    
    # Health check
    health_check uri=/v1/system/health interval=30s;
}

# AWS Application Load Balancer
# Health check path: /v1/system/health
# Health check interval: 30 seconds
# Healthy threshold: 2
# Unhealthy threshold: 5
```

</CodeGroup>

### Monitoring Script

<CodeGroup>

```javascript JavaScript
class HealthMonitor {
  constructor(apiUrl) {
    this.apiUrl = apiUrl;
  }
  
  async checkHealth() {
    try {
      const response = await fetch(`${this.apiUrl}/v1/system/health`);
      const data = await response.json();
      
      return {
        status: data.status,
        services: data.services,
        timestamp: data.timestamp,
        isHealthy: data.status === 'healthy'
      };
    } catch (error) {
      return {
        status: 'unhealthy',
        error: error.message,
        timestamp: new Date().toISOString(),
        isHealthy: false
      };
    }
  }
  
  async startMonitoring(interval = 60000) {
    setInterval(async () => {
      const health = await this.checkHealth();
      
      if (!health.isHealthy) {
        console.warn('System health check failed:', health);
        // Send alert to monitoring system
        this.sendAlert(health);
      }
    }, interval);
  }
  
  sendAlert(health) {
    // Implement alerting logic (email, Slack, PagerDuty, etc.)
    console.log('ALERT: System health issue detected', health);
  }
}

// Usage
const monitor = new HealthMonitor('https://api.railsfromthecrypt.com');
monitor.startMonitoring(60000); // Check every minute
```

</CodeGroup>
