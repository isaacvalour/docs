---
title: "Test Webhook"
api: "POST /v1/webhooks/{id}/test"
description: "Test webhook delivery with a sample event"
---

## Overview

Send a test webhook event to verify your endpoint is working correctly.

## Path Parameters

<ParamField path="id" type="string" required>
  Unique identifier for the webhook
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the test was successful
</ResponseField>

<ResponseField name="response_code" type="number">
  HTTP response code from your endpoint
</ResponseField>

<ResponseField name="response_time" type="number">
  Response time in milliseconds
</ResponseField>

<ResponseField name="message" type="string">
  Test result message
</ResponseField>

<ResponseField name="tested_at" type="string">
  Test timestamp (ISO 8601)
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X POST "https://api.railsfromthecrypt.com/v1/webhooks/wh_123e4567-e89b-12d3-a456-426614174000/test" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.railsfromthecrypt.com/v1/webhooks/wh_123e4567-e89b-12d3-a456-426614174000/test', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_ACCESS_TOKEN'
  }
});

const result = await response.json();
```

</CodeGroup>
