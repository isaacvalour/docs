---
title: "Webhook Status"
api: "GET /v1/webhooks/{id}/status"
description: "Get webhook delivery status and statistics"
---

## Overview

Retrieve webhook delivery status and statistics including success rates, delivery counts, and recent activity.

## Path Parameters

<ParamField path="id" type="string" required>
  Unique identifier for the webhook
</ParamField>

## Response

<ResponseField name="id" type="string">
  Webhook ID
</ResponseField>

<ResponseField name="status" type="string">
  Current webhook status (`active`, `inactive`)
</ResponseField>

<ResponseField name="total_deliveries" type="number">
  Total number of deliveries attempted
</ResponseField>

<ResponseField name="successful_deliveries" type="number">
  Number of successful deliveries
</ResponseField>

<ResponseField name="failed_deliveries" type="number">
  Number of failed deliveries
</ResponseField>

<ResponseField name="success_rate" type="number">
  Success rate percentage (0-100)
</ResponseField>

<ResponseField name="last_delivery" type="string">
  Last delivery timestamp (ISO 8601)
</ResponseField>

<ResponseField name="last_success" type="string">
  Last successful delivery timestamp (ISO 8601)
</ResponseField>

<ResponseField name="last_failure" type="string">
  Last failed delivery timestamp (ISO 8601)
</ResponseField>

<ResponseField name="average_response_time" type="number">
  Average response time in milliseconds
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET "https://api.railsfromthecrypt.com/v1/webhooks/wh_123e4567-e89b-12d3-a456-426614174000/status" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.railsfromthecrypt.com/v1/webhooks/wh_123e4567-e89b-12d3-a456-426614174000/status', {
  headers: {
    'Authorization': 'Bearer YOUR_ACCESS_TOKEN'
  }
});

const status = await response.json();
```

</CodeGroup>
