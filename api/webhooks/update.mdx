---
title: "Update Webhook"
api: "PATCH /v1/webhooks/{id}"
description: "Update webhook settings"
---

## Overview

Update webhook URL, events, or other settings.

## Path Parameters

<ParamField path="id" type="string" required>
  Unique identifier for the webhook
</ParamField>

## Request Body

<ParamField body="url" type="string">
  New webhook URL
</ParamField>

<ParamField body="events" type="array">
  New array of event types
</ParamField>

<ParamField body="secret" type="string">
  New webhook secret
</ParamField>

<ParamField body="active" type="boolean">
  Whether the webhook is active
</ParamField>

## Response

<ResponseField name="id" type="string">
  Webhook ID
</ResponseField>

<ResponseField name="url" type="string">
  Updated webhook URL
</ResponseField>

<ResponseField name="events" type="array">
  Updated event types
</ResponseField>

<ResponseField name="status" type="string">
  Updated status
</ResponseField>

<ResponseField name="updated_at" type="string">
  Update timestamp (ISO 8601)
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X PATCH "https://api.railsfromthecrypt.com/v1/webhooks/wh_123e4567-e89b-12d3-a456-426614174000" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://new-app.com/webhooks/hr",
    "events": ["transaction.completed", "transaction.failed", "transaction.created"]
  }'
```

```javascript JavaScript
const response = await fetch('https://api.railsfromthecrypt.com/v1/webhooks/wh_123e4567-e89b-12d3-a456-426614174000', {
  method: 'PATCH',
  headers: {
    'Authorization': 'Bearer YOUR_ACCESS_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    url: 'https://new-app.com/webhooks/hr',
    events: ['transaction.completed', 'transaction.failed', 'transaction.created']
  })
});

const webhook = await response.json();
```

</CodeGroup>
