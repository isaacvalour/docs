---
title: "Get Webhook"
api: "GET /v1/webhooks/{id}"
description: "Get specific webhook details by ID"
---

## Overview

Retrieve detailed information about a specific webhook using its unique ID.

## Path Parameters

<ParamField path="id" type="string" required>
  Unique identifier for the webhook
</ParamField>

## Response

<ResponseField name="id" type="string">
  Webhook ID
</ResponseField>

<ResponseField name="url" type="string">
  Webhook URL
</ResponseField>

<ResponseField name="events" type="array">
  Subscribed event types
</ResponseField>

<ResponseField name="status" type="string">
  Webhook status (`active`, `inactive`)
</ResponseField>

<ResponseField name="created_at" type="string">
  Creation timestamp (ISO 8601)
</ResponseField>

<ResponseField name="last_delivery" type="string">
  Last successful delivery timestamp (ISO 8601)
</ResponseField>

<ResponseField name="delivery_count" type="number">
  Total number of deliveries attempted
</ResponseField>

<ResponseField name="success_rate" type="number">
  Success rate percentage (0-100)
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET "https://api.railsfromthecrypt.com/v1/webhooks/wh_123e4567-e89b-12d3-a456-426614174000" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.railsfromthecrypt.com/v1/webhooks/wh_123e4567-e89b-12d3-a456-426614174000', {
  headers: {
    'Authorization': 'Bearer YOUR_ACCESS_TOKEN'
  }
});

const webhook = await response.json();
```

</CodeGroup>
