## Overview

Request a password reset email. This endpoint sends a password reset link to the user's email address.

<Note>
  The password reset link will be sent to the user's registered email address.
</Note>

## Request

<ParamField body="email" type="string" required>
  User's email address to send the password reset link to.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the password reset request was successful.
</ResponseField>

<ResponseField name="message" type="string">
  Success message.
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X POST "https://api.railsfromthecrypt.com/v1/auth/forgot-password" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "john.doe@example.com"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.railsfromthecrypt.com/v1/auth/forgot-password', {

  },
```

```python Python

    'https://api.railsfromthecrypt.com/v1/auth/forgot-password',

    }
```

## Notes

- Password reset emails are sent asynchronously
- Reset tokens are typically valid for 1 hour
- Users can only request one password reset per 15 minutes
- The response is the same whether the email exists or not (security measure)
- Check your spam folder if you don't receive the email

## Security Considerations

- The API returns the same response regardless of whether the email exists
- This prevents email enumeration attacks
- Rate limiting prevents abuse of the password reset functionality
- Reset tokens are cryptographically secure and time-limited

## Implementation Example

<CodeGroup>

```javascript JavaScript
class PasswordReset {
  constructor(apiClient) {
    this.apiClient = apiClient;
  }
  
  async requestPasswordReset(email) {
    try {
      const response = await this.apiClient.post('/v1/auth/forgot-password', {
        email: email
      });
      
      if (response.success) {
        // Show success message
        this.showSuccessMessage('Password reset email sent. Please check your inbox.');
        return true;
      } else {
        // Show error message
        this.showErrorMessage(response.message);
        return false;
      }
    } catch (error) {
      this.showErrorMessage('An error occurred while requesting password reset');
      return false;
    }
  }
  
  validateEmail(email) {
    const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    return emailRegex.test(email);
  }
}

// Usage
const passwordReset = new PasswordReset(apiClient);

document.getElementById('forgot-password-form').addEventListener('submit', async (e) => {
  e.preventDefault();
  
  const email = document.getElementById('email').value;
  
  if (passwordReset.validateEmail(email)) {
    await passwordReset.requestPasswordReset(email);
  } else {
    alert('Please enter a valid email address');
  }
});
```

</CodeGroup>
