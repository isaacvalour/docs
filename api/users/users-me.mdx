## Example Request

<CodeGroup>

```bash cURL
curl -X GET https://api.railsfromthecrypt.com/v1/users/me \
  -H "X-API-Key: your-api-key-here"
```

```javascript JavaScript
const response = await fetch('https://api.railsfromthecrypt.com/v1/users/me', {
  method: 'GET',
  headers: {
    'X-API-Key': 'your-api-key-here'
  }
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    'https://api.railsfromthecrypt.com/v1/users/me',
    headers={'X-API-Key': 'your-api-key-here'}
)

data = response.json()
print(data)
```
