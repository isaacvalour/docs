## Overview

Create a new fiat currency recipient that can receive transfers. Fiat recipients are used for traditional banking transfers and require bank account details.

<Note>
Fiat recipients are subject to compliance checks and may require additional verification depending on the amount and destination country.
</Note>

## Request Body

<ParamField body="name" type="string" required>
  Full name of the recipient
  
  **Format:** String (e.g., `"John Doe"`)
  **Validation:** Must be at least 2 characters, maximum 100 characters
</ParamField>

<ParamField body="email" type="string" required>
  Email address of the recipient
  
  **Format:** Valid email address
  **Validation:** Must be a valid email format
</ParamField>

<ParamField body="phone_number" type="string" required>
  Phone number of the recipient
  
  **Format:** International format with country code (e.g., `"+1234567890"`)
  **Validation:** Must include country code
</ParamField>

<ParamField body="country" type="string" required>
  Country code of the recipient
  
  **Format:** ISO 3166-1 alpha-2 country code (e.g., `"US"`, `"NG"`)
  **Validation:** Must be a valid country code
</ParamField>

<ParamField body="currency" type="string" required>
  Currency code for the recipient
  
  **Format:** ISO 4217 currency code (e.g., `"USD"`, `"NGN"`, `"EUR"`)
  **Validation:** Must be a supported currency
</ParamField>

<ParamField body="bank_name" type="string" required>
  Name of the recipient's bank
  
  **Format:** String (e.g., `"First Bank of Nigeria"`)
  **Validation:** Must be at least 2 characters
</ParamField>

<ParamField body="account_number" type="string" required>
  Bank account number
  
  **Format:** String (e.g., `"1234567890"`)
  **Validation:** Must be alphanumeric, length varies by country
</ParamField>

<ParamField body="routing_number" type="string">
  Bank routing number (US) or sort code (UK)
  
  **Format:** String (e.g., `"021000021"`)
  **Note:** Required for US and UK banks
</ParamField>

<ParamField body="swift_code" type="string">
  SWIFT/BIC code for international transfers
  
  **Format:** 8-11 character SWIFT code (e.g., `"BOFAUS3N"`)
  **Note:** Required for international transfers
</ParamField>

<ParamField body="iban" type="string">
  International Bank Account Number
  
  **Format:** Valid IBAN format
  **Note:** Required for European countries
</ParamField>

<ParamField body="address" type="object">
  Physical address of the recipient
</ParamField>

<ParamField body="address.street" type="string">
  Street address
</ParamField>

<ParamField body="address.city" type="string">
  City name
</ParamField>

<ParamField body="address.state" type="string">
  State or province
</ParamField>

<ParamField body="address.postal_code" type="string">
  Postal or ZIP code
</ParamField>

<ParamField body="address.country" type="string">
  Country code (should match recipient country)
</ParamField>

<ParamField body="metadata" type="object">
  Additional metadata for the recipient
  
  **Format:** JSON object with custom fields
  **Note:** Optional field for storing additional context
</ParamField>

### Create Recipient with Validation

<CodeGroup>

```javascript JavaScript
class FiatRecipientManager {
  constructor(apiClient) {
    this.apiClient = apiClient;
  }
  
  async createRecipient(recipientData) {
    try {
      // Validate recipient data
      this.validateRecipientData(recipientData);
      
      // Create recipient
      const response = await fetch('https://api.railsfromthecrypt.com/v1/recipients/fiat', {
        method: 'POST',
        headers: {
          'Authorization': `Bearer ${accessToken}`,
          'Content-Type': 'application/json'
        },
        body: JSON.stringify(recipientData)
      });

      if (!response.ok) {
        const error = await response.json();
        throw new Error(error.message || 'Failed to create recipient');
      }

      const recipient = await response.json();
      
      // Store recipient for future use
      this.storeRecipient(recipient);
      
      return recipient;
    } catch (error) {
      console.error('Error creating fiat recipient:', error);
      throw error;
    }
  }
  
  validateRecipientData(data) {
    const required = ['name', 'email', 'phone_number', 'country', 'currency', 'bank_name', 'account_number'];
    
    for (const field of required) {
      if (!data[field]) {
        throw new Error(`Missing required field: ${field}`);
      }
    }
    
    // Validate email format
    const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    if (!emailRegex.test(data.email)) {
      throw new Error('Invalid email format');
    }
    
    // Validate phone number format
    const phoneRegex = /^\+[1-9]\d{1,14}$/;
    if (!phoneRegex.test(data.phone_number)) {
      throw new Error('Invalid phone number format');
    }
    
    // Validate country-specific requirements
    if (data.country === 'US' && !data.routing_number) {
      throw new Error('Routing number required for US recipients');
    }
    
    if (data.country === 'FR' && !data.iban) {
      throw new Error('IBAN required for French recipients');
    }
  }
  
  storeRecipient(recipient) {
    // Store recipient in local storage or state management
    const recipients = JSON.parse(localStorage.getItem('recipients') || '[]');
    recipients.push(recipient);
    localStorage.setItem('recipients', JSON.stringify(recipients));
  }
}

// Usage
const recipientManager = new FiatRecipientManager();
await recipientManager.createRecipient(recipientData);
```
