## Overview

Create a new client account for business entities. This endpoint is used to set up business clients that can use the HR API services.

<Note>
  This endpoint requires authentication via Bearer token in the Authorization header.
</Note>

## Implementation Example

<CodeGroup>

```javascript JavaScript
class ClientManager {
  constructor(apiClient) {
    this.apiClient = apiClient;
  }
  
  async createClient(clientData) {
    try {
      const response = await this.apiClient.post('/v1/client', clientData);
      
      if (response.success) {
        console.log('Client created successfully:', response.data);
        return response.data;
      } else {
        throw new Error(response.message);
      }
    } catch (error) {
      console.error('Failed to create client:', error);
      throw error;
    }
  }
  
  validateClientData(clientData) {
    const errors = [];
    
    if (!clientData.company_name) {
      errors.push('Company name is required');
    }
    
    if (!clientData.company_email) {
      errors.push('Company email is required');
    } else if (!this.isValidEmail(clientData.company_email)) {
      errors.push('Invalid company email format');
    }
    
    if (!clientData.company_phone_number) {
      errors.push('Company phone number is required');
    } else if (!this.isValidPhoneNumber(clientData.company_phone_number)) {
      errors.push('Invalid phone number format');
    }
    
    if (!clientData.company_country) {
      errors.push('Company country is required');
    }
    
    if (clientData.authorized_signatory) {
      const signatory = clientData.authorized_signatory;
      if (!signatory.first_name) {
        errors.push('Signatory first name is required');
      }
      if (!signatory.last_name) {
        errors.push('Signatory last name is required');
      }
      if (!signatory.email) {
        errors.push('Signatory email is required');
      } else if (!this.isValidEmail(signatory.email)) {
        errors.push('Invalid signatory email format');
      }
    }
    
    return {
      isValid: errors.length === 0,
      errors: errors
    };
  }
  
  isValidEmail(email) {
    const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    return emailRegex.test(email);
  }
  
  isValidPhoneNumber(phoneNumber) {
    const phoneRegex = /^\+[1-9]\d{1,14}$/;
    return phoneRegex.test(phoneNumber);
  }
  
  async createClientWithValidation(clientData) {
    const validation = this.validateClientData(clientData);
    
    if (!validation.isValid) {
      throw new Error(`Validation failed: ${validation.errors.join(', ')}`);
    }
    
    return this.createClient(clientData);
  }
}

// Usage
const clientManager = new ClientManager(apiClient);

const clientData = {
  company_name: 'Acme Corporation',
  company_email: 'contact@acme.com',
  company_phone_number: '+1234567890',
  company_country: 'US',
  company_address: {
    street: '123 Business St',
    city: 'New York',
    state: 'NY',
    postal_code: '10001'
  },
  business_type: 'corporation',
  registration_number: '123456789',
  tax_id: '12-3456789',
  website: 'https://acme.com',
  description: 'A leading technology company',
  authorized_signatory: {
    first_name: 'John',
    last_name: 'Doe',
    email: 'john.doe@acme.com',
    phone_number: '+1234567890',
    position: 'CEO'
  }
};

try {
  const client = await clientManager.createClientWithValidation(clientData);
  console.log('Client created:', client);
} catch (error) {
  console.error('Failed to create client:', error.message);
}
```

## KYB Verification Process

After creating a client, the Know Your Business (KYB) verification process begins:

<Steps>
  <Step title="Client Created">
    Client account is created with pending KYB status
  </Step>
  <Step title="Document Upload">
    Required business documents are uploaded
  </Step>
  <Step title="Verification Review">
    Documents are reviewed by compliance team
  </Step>
  <Step title="Approval">
    Client is approved and can use services
  </Step>
</Steps>
