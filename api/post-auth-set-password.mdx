## Overview

Set initial password for account

<Note>
This endpoint requires authentication. Include your API key in the `X-API-Key` header or use Bearer token authentication.
</Note>

## Request

<ParamField body="password" type="string" required>
  New password
  
  **Example:** `StrongP@ssw0rd!`
</ParamField>

<ParamField body="confirm_password" type="string" required>
  Confirm new password
  
  **Example:** `StrongP@ssw0rd!`
</ParamField>

## Example Request

<CodeGroup>

```bash cURL
curl -X POST \
  'https://api.railsfromthecrypt.com/v1/auth/set-password' \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -d '{"password": "StrongP@ssw0rd!", "confirm_password": "StrongP@ssw0rd!"}'
```

```javascript JavaScript
const response = await fetch('https://api.railsfromthecrypt.com/v1/auth/set-password', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_TOKEN'
  },
  body: JSON.stringify({"password": "StrongP@ssw0rd!", "confirm_password": "StrongP@ssw0rd!"})
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

url = 'https://api.railsfromthecrypt.com/v1/auth/set-password'
headers = {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_TOKEN'
}
data = {"password": "StrongP@ssw0rd!", "confirm_password": "StrongP@ssw0rd!"}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(result)
```
