---
title: "List Transfer Requests"
api: "GET /v1/transfer-requests"
description: "Get list of all transfer requests with filtering and pagination"
---

## Overview

Retrieve a paginated list of all transfer requests with optional filtering by status, date range, and other criteria.

## Query Parameters

<ParamField query="limit" type="number">
  Number of requests to return (default: 20, max: 100)
</ParamField>

<ParamField query="offset" type="number">
  Number of requests to skip (default: 0)
</ParamField>

<ParamField query="status" type="string">
  Filter by status (`pending`, `processing`, `completed`, `failed`, `cancelled`)
</ParamField>

<ParamField query="currency" type="string">
  Filter by source currency
</ParamField>

<ParamField query="recipient_id" type="string">
  Filter by recipient ID
</ParamField>

<ParamField query="created_after" type="string">
  Filter requests created after this date (ISO 8601)
</ParamField>

<ParamField query="created_before" type="string">
  Filter requests created before this date (ISO 8601)
</ParamField>

<ParamField query="sort" type="string">
  Sort order (`created_at`, `-created_at`, `amount`, `-amount`)
</ParamField>

## Response

<ResponseField name="transfer_requests" type="array">
  Array of transfer request objects
</ResponseField>

<ResponseField name="transfer_requests[].id" type="string">
  Transfer request ID
</ResponseField>

<ResponseField name="transfer_requests[].reference" type="string">
  Transfer reference
</ResponseField>

<ResponseField name="transfer_requests[].status" type="string">
  Transfer status
</ResponseField>

<ResponseField name="transfer_requests[].amount" type="string">
  Transfer amount
</ResponseField>

<ResponseField name="transfer_requests[].currency" type="string">
  Source currency
</ResponseField>

<ResponseField name="transfer_requests[].recipient" type="object">
  Recipient information
</ResponseField>

<ResponseField name="transfer_requests[].created_at" type="string">
  Creation timestamp (ISO 8601)
</ResponseField>

<ResponseField name="pagination" type="object">
  Pagination information
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
# Get all transfer requests
curl -X GET "https://api.railsfromthecrypt.com/v1/transfer-requests" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"

# Get pending requests only
curl -X GET "https://api.railsfromthecrypt.com/v1/transfer-requests?status=pending" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.railsfromthecrypt.com/v1/transfer-requests', {
  headers: {
    'Authorization': 'Bearer YOUR_ACCESS_TOKEN'
  }
});

const data = await response.json();
```

</CodeGroup>
