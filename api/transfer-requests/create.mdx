## Overview

Create a new transfer request to send funds to a recipient. This is the recommended way to initiate transfers as it provides a streamlined flow with automatic rate calculation and payment instructions.

<Note>
Transfer requests are initially in `pending` status and require payment confirmation before processing. Use the confirm-payment endpoint once payment is made.
</Note>

## Request Body

<ParamField body="amount" type="string" required>
  Amount to transfer in source currency
</ParamField>

<ParamField body="currency" type="string" required>
  Source currency code (e.g., `"USD"`, `"NGN"`)
</ParamField>

<ParamField body="recipient_id" type="string" required>
  ID of the recipient to send funds to
</ParamField>

<ParamField body="rate" type="string">
  Exchange rate to use (if not provided, current market rate will be used)
</ParamField>

<ParamField body="pair" type="string">
  Currency pair (e.g., `"USD/USDC"`)
</ParamField>

<ParamField body="received_amount" type="string">
  Expected amount recipient will receive
</ParamField>

<ParamField body="fee" type="string">
  Transfer fee amount
</ParamField>

<ParamField body="reference" type="string">
  Custom reference for this transfer
</ParamField>

<ParamField body="description" type="string">
  Description or memo for this transfer
</ParamField>

<ParamField body="metadata" type="object">
  Additional metadata for this transfer
</ParamField>

## Example Request

<CodeGroup>

```bash cURL
curl -X POST "https://api.railsfromthecrypt.com/v1/transfer-requests" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": "1000.00",
    "currency": "NGN",
    "recipient_id": "rec_123e4567-e89b-12d3-a456-426614174000",
    "description": "Payment for services"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.railsfromthecrypt.com/v1/transfer-requests', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_ACCESS_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    amount: '1000.00',
    currency: 'NGN',
    recipient_id: 'rec_123e4567-e89b-12d3-a456-426614174000',
    description: 'Payment for services'
  })
});

const transferRequest = await response.json();
```

## Usage Examples

### Create Transfer Request

<CodeGroup>

```javascript JavaScript
async function createTransferRequest(transferData) {
  try {
    const response = await fetch('https://api.railsfromthecrypt.com/v1/transfer-requests', {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${accessToken}`,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify(transferData)
    });

    if (!response.ok) {
      const error = await response.json();
      throw new Error(error.message || 'Failed to create transfer request');
    }

    const transferRequest = await response.json();
    return transferRequest;
  } catch (error) {
    console.error('Error creating transfer request:', error);
    throw error;
  }
}

// Usage
const transferData = {
  amount: '1000.00',
  currency: 'NGN',
  recipient_id: 'rec_123e4567-e89b-12d3-a456-426614174000',
  description: 'Payment for services'
};

const transferRequest = await createTransferRequest(transferData);
```

</CodeGroup>
