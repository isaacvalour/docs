---
title: "Get IP Address"
api: "GET /v1/ips/{id}"
description: "Get specific IP address details by ID"
---

## Overview

Retrieve detailed information about a specific whitelisted IP address using its unique ID.

## Path Parameters

<ParamField path="id" type="string" required>
  Unique identifier for the IP address
</ParamField>

## Response

<ResponseField name="id" type="string">
  IP address ID
</ResponseField>

<ResponseField name="ip_address" type="string">
  Whitelisted IP address or range
</ResponseField>

<ResponseField name="label" type="string">
  IP address label
</ResponseField>

<ResponseField name="description" type="string">
  IP address description
</ResponseField>

<ResponseField name="status" type="string">
  IP address status (`active`, `inactive`)
</ResponseField>

<ResponseField name="created_at" type="string">
  Creation timestamp (ISO 8601)
</ResponseField>

<ResponseField name="updated_at" type="string">
  Last update timestamp (ISO 8601)
</ResponseField>

<ResponseField name="last_used" type="string">
  Last usage timestamp (ISO 8601)
</ResponseField>

<ResponseField name="usage_count" type="number">
  Number of times this IP has been used
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET "https://api.railsfromthecrypt.com/v1/ips/ip_123e4567-e89b-12d3-a456-426614174000" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.railsfromthecrypt.com/v1/ips/ip_123e4567-e89b-12d3-a456-426614174000', {
  headers: {
    'Authorization': 'Bearer YOUR_ACCESS_TOKEN'
  }
});

const ipAddress = await response.json();
```

</CodeGroup>
