---
title: "Delete IP Address"
api: "DELETE /v1/ips/{id}"
description: "Remove IP address from whitelist"
---

## Overview

Remove an IP address from the whitelist. This action cannot be undone.

<Warning>
Removing an IP address from the whitelist will immediately block requests from that IP when IP whitelisting is enabled.
</Warning>

## Path Parameters

<ParamField path="id" type="string" required>
  Unique identifier for the IP address to delete
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the deletion was successful
</ResponseField>

<ResponseField name="message" type="string">
  Confirmation message
</ResponseField>

<ResponseField name="deleted_at" type="string">
  Deletion timestamp (ISO 8601)
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X DELETE "https://api.railsfromthecrypt.com/v1/ips/ip_123e4567-e89b-12d3-a456-426614174000" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.railsfromthecrypt.com/v1/ips/ip_123e4567-e89b-12d3-a456-426614174000', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_ACCESS_TOKEN'
  }
});

const result = await response.json();
```

</CodeGroup>

## Example Response

<ResponseExample>
