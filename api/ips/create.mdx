## Overview

Add a new IP address or IP range to the whitelist for enhanced security. Only requests from whitelisted IPs will be allowed when IP whitelisting is enabled.

<Note>
IP whitelisting is an optional security feature. When enabled, only requests from whitelisted IPs will be processed.
</Note>

## Request Body

<ParamField body="ip_address" type="string" required>
  IP address or CIDR range to whitelist (e.g., `"192.168.1.1"` or `"192.168.1.0/24"`)
</ParamField>

<ParamField body="label" type="string" required>
  Human-readable label for the IP address
</ParamField>

<ParamField body="description" type="string">
  Optional description for the IP address
</ParamField>

<ParamField body="active" type="boolean">
  Whether the IP address is active (default: true)
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique identifier for the IP address
</ResponseField>

<ResponseField name="ip_address" type="string">
  Whitelisted IP address or range
</ResponseField>

<ResponseField name="label" type="string">
  IP address label
</ResponseField>

<ResponseField name="description" type="string">
  IP address description
</ResponseField>

<ResponseField name="status" type="string">
  IP address status (`active`, `inactive`)
</ResponseField>

<ResponseField name="created_at" type="string">
  Creation timestamp (ISO 8601)
</ResponseField>

<ResponseField name="last_used" type="string">
  Last usage timestamp (ISO 8601)
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X POST "https://api.railsfromthecrypt.com/v1/ips" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "ip_address": "192.168.1.100",
    "label": "Office Network",
    "description": "Main office IP address"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.railsfromthecrypt.com/v1/ips', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_ACCESS_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    ip_address: '192.168.1.100',
    label: 'Office Network',
    description: 'Main office IP address'
  })
});

const ipAddress = await response.json();
```

</CodeGroup>
