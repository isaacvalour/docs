## Overview

Register a new user account with company information

## Request

<ParamField body="email" type="string" required>
  User email address
  
  **Example:** `user@example.com`
</ParamField>

<ParamField body="invitation_code" type="string" >
  Invitation code (optional)
  
  **Example:** `123456`
</ParamField>

## Example Request

<CodeGroup>

```bash cURL
curl -X POST \
  'https://api.railsfromthecrypt.com/v1/auth/register' \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -d '{"email": "user@example.com", "invitation_code": "123456"}'
```

```javascript JavaScript
const response = await fetch('https://api.railsfromthecrypt.com/v1/auth/register', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_TOKEN'
  },
  body: JSON.stringify({"email": "user@example.com", "invitation_code": "123456"})
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

url = 'https://api.railsfromthecrypt.com/v1/auth/register'
headers = {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_TOKEN'
}
data = {"email": "user@example.com", "invitation_code": "123456"}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(result)
```
